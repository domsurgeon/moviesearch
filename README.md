# Moviesearch - App to search for movies

## Requirement

- NPM / Yarn
- Node

## Usage

Clone it!

```
$ git clone git@github.com:domsurgeon/moviesearch.git
```

Go into the project directory and run the command:

```
$ yarn install
$ yarn start
```

Navigator shoul open!
